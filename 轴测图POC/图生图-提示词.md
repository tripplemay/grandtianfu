# 阅天府 D户型 · 轴测图生图（img2img）操作包

目标：**用我们生成的精确轴测图当底图，做写实 dollhouse，几何锁死不漂移。**
gateway 做不了图生图（`generate_image` 无图片入口、`seedream-4-5` 不支持图片输入），需用下列外部工具。

## 底图文件（二选一/都可）

| 底图 | 适用 | 特点 |
|---|---|---|
| `轴测效果图-精细版-超清3600.png` | Nano Banana / Gemini / gpt-image | 彩色，自带材质/家具暗示，模型更易理解 |
| `轴测线稿净版.png` | ControlNet（canny/lineart/mlsd） | 纯黑线白底、隐藏线已消除，**锁结构最强** |

## 方案一：Nano Banana / Google Gemini（最省事，你已在用）

**用法**：在 Google AI Studio / Gemini 里上传底图（用彩色超清版），粘贴下面提示词运行（image editing / img2img）。

**Prompt（英文，效果最佳）：**
```
Using the provided isometric cutaway floor-plan image as the EXACT structural reference,
render a photorealistic 3D isometric dollhouse of this same apartment.

CRITICAL — keep the identical layout: same outer shape, same wall positions, same room
divisions, and the same furniture types and placements as in the reference. Do NOT add,
remove, move, merge or resize any walls or furniture. Keep the same top-down 45° axonometric
angle and the same low cut-away walls.

Only convert it into a realistic architectural miniature with real materials, textures, soft
global-illumination daylight and subtle contact shadows.

Materials (modern light-luxury / 现代轻奢):
- warm walnut wood plank floors; walnut slatted TV feature wall
- beige travertine stone; white marble countertops and coffee table
- matte micro-cement walls in bathrooms; off-white painted walls elsewhere
- brushed brass / antique-bronze accents
- caramel leather L-shaped sofa; dark-green velvet swivel armchairs; cream & camel textiles
- leafy green plants; floor-to-ceiling glass with a soft, blurred city view
Clean light-grey studio background, ultra-detailed, photorealistic.
```
**中文补充指令（可加在末尾）：** `严格保持原图的房间布局、墙体与家具位置不变，只做写实材质与灯光。`

> 若结构仍漂移：换用 `轴测线稿净版.png` 当底图，并强调 “follow the line drawing exactly”。

## 方案二：ControlNet（锁结构最强，SDXL / Flux）

**底图**：`轴测线稿净版.png`
**跑法**：ComfyUI / A1111（本地免费），或 Replicate / fal.ai（API 按次付费）。

**推荐配置**
- 基础模型：SDXL 写实大模型（RealVisXL / Juggernaut XL）或 Flux.1-dev
- ControlNet ①：`lineart`（或 `canny` / `mlsd`，mlsd 对直墙最稳）
  - control weight **0.85–1.0**，start 0.0，end 0.9
- ControlNet ②（可选，增体积感）：`depth`，weight 0.4–0.6
- 采样：steps 30，CFG 5–7
- 模式：纯 txt2img + ControlNet（denoise 由 control 主导）；若用 img2img 起稿则 denoise 0.6–0.75
- 分辨率：按底图长宽比，建议长边 ≥1536；不够清晰用 tiled upscale

**Positive prompt：**
```
photorealistic isometric dollhouse cutaway of a modern light-luxury apartment, architectural
miniature model, no roof, top-down 45 degree axonometric view, warm walnut wood plank floors,
walnut slatted tv feature wall, beige travertine stone, white marble countertops, caramel
leather L-shaped sofa, dark green velvet swivel armchairs, brushed brass accents, central
kitchen island, freestanding bathtub, double vanity, glass shower, leafy plants, floor to
ceiling windows with soft blurred city view, soft natural daylight, global illumination,
ultra detailed, 8k, professional interior design render
```

**Negative prompt：**
```
flat, 2d, cartoon, sketch, low quality, blurry, distorted geometry, warped walls, extra walls,
missing walls, merged rooms, wrong perspective, fisheye, text, watermark, people, oversaturated
```

**调参经验**
- 几何漂移 → 提高 ControlNet weight（→1.0）、用 `mlsd`、加 `depth` 第二控；
- 太死板/塑料感 → 适度降 weight（0.8）、提高 CFG 或加细节 LoRA；
- 想换视角/层高 → 改 `轴测线稿-生成器.py` 里投影角度或 `WALL_H` 后重出线稿。

## 重出底图（改平面后）
1. 先改 `平面布置图-生成器.py` → 跑出新 `平面布置图.svg`(含家具) **和** `平面布置图-无家具.svg`(干净底图，方案四用)
2. 彩色精细版：`python3 轴测图POC/轴测图-生成器-精细版.py` → rsvg-convert 转 PNG
3. 线稿净版：`python3 轴测图POC/轴测线稿-生成器.py` → rsvg-convert 转 PNG
（坐标系 1px=10mm，脚本内高度用 mm、投影 ZK=0.1 已处理）
