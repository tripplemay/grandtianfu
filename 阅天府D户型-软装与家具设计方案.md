## 一、硬装基底分析（软装的设计前提）

实景视频显示本房为**高端精装交付**，硬装已经把整体基调锁定在「暖调现代轻奢」，软装必须顺色系延展、不可跳脱。现有基底要素：

| 区域 | 现有硬装材质 | 对软装的约束 |
|---|---|---|
| 公区地面 | 米色洞石纹大理石地砖（暖灰米，带流纹） | 地毯需压住大面积冷光泽，用暖色羊毛地毯 |
| 公区墙面 | 灰咖色质感墙漆 + **深胡桃木饰面**门套/墙体 | 家具木色统一向胡桃木靠拢；墙面挂画需提亮 |
| 公区造景 | 书纹白大理石造景墙 + 铜色收边 + 顶面灯带 | 金属饰品/灯具统一用**香槟金/古铜**，忌冷不锈钢 |
| 主卧 | 暖橡木地板 + **奶咖色木饰面格栅**背景墙 | 床品走奶咖/燕麦色系，与格栅同温 |
| 主卫 | 金棕大理石(类玉石)双台盆 + 微水泥灰墙 + 独立浴缸 + LED灯镜 | 仅做软装点缀（毛巾、托盘、香薰、绿植） |
| 厨房 | 浅灰烤漆橱柜 + 白石英石台面 + 黑框玻璃移门 | 餐区五金/餐椅可呼应黑框线条 |
| 次卧/客房 | 白墙 + 暖木色地板 + 中央空调 + 局部灰色质感壁布 | 床头背景靠软装（床头板/挂画/壁灯）出彩 |
| 全屋采光 | 大面落地窗，城市+中庭双景观 | 窗帘做"纱+遮光"双层，白天保留通透 |

**结论**：硬装造价不低、完成度高，软装策略是 **「克制、提质、点睛」**——不做与硬装争抢的造型，把预算压在"人能接触到/天天用到"的核心件（床、主沙发、餐桌椅、地毯、窗帘、主灯）上，用材质和工艺拉高级感。

## 三、设计风格与色彩材质策略

### 风格关键词
**现代轻奢 · 暖调静奢（Quiet Luxury）**——低饱和、重材质、弱造型、强质感。延续硬装的胡桃木+金铜+石材，软装注入丝绒、半哑皮革、羊毛、亚麻与艺术玻璃，做"看不出贵但处处是料"的高级感。

### 全屋色彩体系（统一控色，避免每间各自为政）
- **主色（70%）**：暖白 / 燕麦米 / 浅灰咖（墙、顶、大面布艺）
- **辅色（25%）**：胡桃木棕 / 焦糖驼色 / 雾霾灰蓝（家具、皮革、单椅）
- **点缀色（5%）**：香槟金 / 古铜 / 墨绿 / 酒红（金属、抱枕、艺术品、绿植）

### 核心材质清单
丝绒、半哑光真皮（Nappa/植鞣）、羊毛+真丝地毯、亚麻棉麻窗纱、胡桃木实木/木皮、艺术微水泥、雾面玻璃、香槟金不锈钢、天然大理石/洞石、藤编与陶艺（自然点缀）。

## 五、灯光（软装可调部分）

硬装已有筒灯+灯带+中央空调风口，软装层补充**情绪光与重点光**：
- 全屋光源色温统一 **2700–3000K 暖光**，主灯/落地/台灯**全部可调光**。
- 客厅：落地灯+台灯+灯带，避免只开主光的"工装感"。
- 餐厅：艺术吊灯为视觉中心，配可调光。
- 卧室：床头壁灯 + 氛围地脚灯 + 衣帽间感应灯。
- 全屋可接入智能场景（回家/会客/观影/睡眠四场景），用米家/Aqara/控客等。

## 六、布艺 · 窗帘 · 地毯（统一定调，单独招商）
- **窗帘**：全屋"纱+遮光"双层，色走燕麦/浅咖；卧室高遮光，公区重通透；电动轨道（落地窗位）。
- **地毯**：客厅羊毛手工毯为重点件，卧室羊毛，书房短绒吸音；统一暖灰咖色域、深浅区分空间。
- **抱枕/床品/桌旗**：全屋按"主色+辅色+1点缀"三色规则配比，材质做丝绒/棉麻/针织/皮revers 的层次混搭。

## 八、艺术品 · 绿植 · 饰品陈设原则
- **艺术品**：每空间 1 处视觉主画/主雕塑 + 若干小品；色彩从全屋点缀色（金/绿/酒红）取，呼应而不喧宾。
- **真植**：公区大型（琴叶榕/天堂鸟/橄榄树），卧室卫浴中小型（散尾葵/龟背竹/虎皮兰），统一陶/水泥/藤编花器。
- **饰品**："成组、三角构图、留白"三原则；材质混搭金属+陶艺+书籍+蜡烛，忌平铺堆砌。

---

## 九、落地与采购建议（执行节奏）
1. **先定大件主基调**：主沙发、主床、餐桌椅、客厅地毯——这四组决定全屋气质，先选定再向外延展配色。
2. **窗帘与灯光同步**：窗帘是大面积色块，需与沙发/床品同步打样比色；移动灯具同期定。
3. **核心件下单周期长**：进口/定制沙发、床垫、整墙书柜需 30–60 天，**优先下单**。
4. **饰品艺术品最后软陈设**：家具进场后再现场配饰品、花艺、挂画，按实际光影与留白微调。
5. **建议出"软装意向板(Mood Board)+ 3D 软装效果图"** 与本清单配套确认后再采购，避免色差与比例误差。
