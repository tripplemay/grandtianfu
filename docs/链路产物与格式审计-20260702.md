- 审计日期:2026-07-02(生产 tag `aa35c53`)

- 范围:每步产物是否承载下一步所需信息、文件格式是否适合传递、优化点

## 总评

链路骨架健康:geometry 单一真源 + 派生不落盘、家具 room-relative 相对坐标、scheme↔baseline 版本门禁、产物 uuid 不可变归档、SVG 不进产物目录的 XSS 白名单 —— 核心格式决策正确。**问题集中在交接处的「信息保真」**:

- 最大错配在**第 7 步输入侧**:jpg/webp 硬标 `image/png`、EXIF 方向不重写、竖拍被压横幅、整宅轴测对单间照片 —— 四个问题同时对抗提示词「保持相机与几何不变」的核心意图。
- 第二错配在 **4→5 栅格**:底图纵横比随户型 bbox 变化(D 实测 1536×1091≈1.408)却固定送 edits `1536x1024`(1.5),模型被迫重取景。
- 两类贯穿缺口:**复现链断裂**(底图 PNG 与 prompt 原文不归档、记录无 created_at/engine_version)与**持久层盲区**(artifacts 效果图与 uploads 客户照片零备份、无 GC)。
- 新发现:**style_prompt 在第 2 步后断链** —— 第 5 步提示词硬编码「现代轻奢」,用户风格意向不进效果图。

种子问题 6 条核实 5 条;「renders.json 膨胀」被高估(manifest 为 hash 摘要 ~950B/条,cap 满 ~185KB;真问题是响应载荷 3/4 前端零消费 + 无缩略图)。

## 优化清单

### P0(格式错配/直接影响出图质量)
1. **输入图 mime 修正**(S):provider.edit 按 magic bytes 判型(PNG/JPEG/WEBP)设文件名与 mime,未识别拒绝。`providers.py:73-74`
2. **上传归一化管线**(M):Pillow verify → `exif_transpose` → 转 RGB JPEG q90(天然剥 EXIF 含 GPS)→ 最长边 2048 → 写 width/height/mime/sha256 进 photos.json;pillow-heif 支持 HEIC。
3. **第 7 步轴测按房切片**(M,Phase 1.5c 补课):`slice_*_for_room(room_id)` 收窄参考图(rooms/walls/doors/furniture 按目标房过滤);photo.room_id 空则回退整宅;record 记 axon_scope。可行性已确认(axon 纯数据驱动、axon_furniture 每件保有 _room_id)。
4. **4→5 栅格纵横比对齐**(M):按 SVG viewBox 比例在 {1024x1024, 1536x1024, 1024x1536} 选最近档,letterbox 到目标画布后送 edits 同 size(栅格与 edit 尺寸单一来源)。
5. **第 7 步输出 size 按照片比例三档选**(S):h/w>1.2→1024x1536,0.8-1.2→1024x1024,否则 1536x1024。
6. **style_prompt 贯通第 5 步**(S):`prompt_gen.generate(style=...)`,None 时字节输出不变(保护金测)。
7. **备份覆盖 artifacts/uploads**(S):backup.sh 上提到 data 整目录或三目录各一包;restore-test 同步;uploads 含用户照片建议加密。

### P1(信息缺失/可追溯)
底图+prompt 原文归档 + created_at/engine_version;mode 受控枚举+前端正向过滤(切片新 mode 前置);furniture 写边界 item 校验(400+index);派生补口(passage 进派生/推拉门宽兜底/orient 朝向/避落地窗);photos 元数据+direction 白名单+未标注拦截;单位自描述(catalog docstring 修正 + meta units 块 + axon 断言 mm_per_px==10);plan2d SVG 文本转义+下载改 PNG;prompt 方位改用调整后坐标。

### P2(工程)
JSON 版本闸门(读方 dict 分支+未知 schema 显式抛错);产物 GC(mark-and-sweep 脚本+uploads 配额+裸 /uploads 端点退役);renders 读侧瘦身(list 剥 manifest/usage + limit)+全链缩略图(320px webp);legacy 收口(RenderLog 双址迁移/根级镜像退役/default meta 双规范化统一/fsync 补齐);契约小项(SchemeSource 补 migrated/死信封清理/悬挂件表现统一/绝对件后门收口);furnish model/警告落 meta + catalog CATALOG_REV。
