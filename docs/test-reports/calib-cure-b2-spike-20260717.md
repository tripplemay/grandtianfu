`solve_pnp_general`（通用 PnP：焦距扫描 + 已知 K 位姿归一化 DLT + 正交化 + 物理门 + 最小重投影 → Gauss-Newton 精修）合成真值自检：

- LM 阻尼 + log_f 钳制后无数值溢出；退化共线输入给解不崩（由 F004 守门在信任前拦）。

### 1.2 异面点破共面退化的鲁棒性增益 ✅（合成台）
真值相机 + σ=8px 点击噪声，失败=相机中心误差>0.5m：

| 点集 | 现产品 `solve_pnp`（共面单应） | 新 `solve_pnp_general` |

| 纯地面 4 点（共面） | — | 13% 失败 |

### 1.3 真实病例照片的特征可见性 ✅（目检 2 张生产空房照）
- **472015c4（小卧室/书房，798 案同型）**：清晰地面墙角 ~2 个，但**有一条完整竖直墙棱**（窗墙×右墙交角，地面→天花板贯通）天然给异面点对；另有 2–3 个天花板-墙交角。→ 地面+竖直+天花板合计 ~5–6 个可标点，跨两高度。

## 2. 验准的配方（Planner 据此锁 F002–F007）

1. **F002 特征派生富化**：现只派生地面 z=0 点。**必加**：墙-天花板角（z=2700）、墙角竖直棱（z=0..2700，给异面约束）、门/窗竖框顶点、全高窗竖挺。真实照片证明这些是可标的。
2. **F003 通用 PnP + GN 精修**：原型 `solve_pnp_general` 即范式（焦距扫描 + 已知 K 位姿 DLT + 正交化 + 物理门 + GN 精修）。产品化时保合成真值往返 <2px、共面输入仍可解（退化到现路径或统一新路径）。
3. **F004 退化/可见性守门**：解算前查点位条件数（`degeneracy()`：collinear_ratio + height_span_mm + n_points）。近共线 / 全在单一高度 / 点太少 → 明确引导（"点一个天花板角把点铺到两个高度""点竖直墙棱的上下两端"）。
4. **F005 稳健 reproj 指标**：max→RMS/分位，门仍诚实。
5. **F006 前端多高度点选**：地面 + 天花板/竖直点点选 + 铺开引导（呼应 F004）。
6. **F007 专家模式降级**：合成台 σ=3px→55% 撞门，最脆，降级为高级选项。
7. **补充建议（Planner 裁决是否纳入本批或立项）**：
