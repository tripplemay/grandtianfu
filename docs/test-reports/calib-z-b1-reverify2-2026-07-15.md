### 3.5 但**结论仍然成立** —— 因为有一条更强、且不依赖照片/精度的判据

我实测**全部 5 张 / 11 条**，修复后物理门（锚点在前方 + 相机在地板上方）之后：

```
1537e : 存活候选 = 1  ->  det=-1  C=(14463.3,  4685.5, 1515.1)   锚点err(sum)=195.7

```

**关键推理（对精度完全不敏感）：**

1. 世界系 (East, South, Up) 是**左手系**，相机系是右手系 ⇒ 物理可实现的 world→camera `R` **必然 det = −1**（§2 合成真值换基推导已独立证明）。
2. ⇒ 生产在用的 11 条**全部 det = +1 = 手性错误 = 成像被镜像 = 物理不可实现**。这不是「可能不对」，是**可证伪地不对**。
3. 修复后物理门筛完，**11/11 恰好只剩 1 个候选**，且它 det = −1。
4. 1537e 的 stored 与 healed 是**精确镜像对**，对锚点的重投影 err **完全相同**（我实测两者 max 均 123.9 / sum 均 195.7）—— 即**数据本身对二者无偏好**，方向只能由物理定，而物理只允许 det = −1 的那个。

⇒ **healing 1537e 不是「赌一个方向」，而是把一台可证伪为不可实现的镜像相机，换成唯一物理可容许的解。「healed 严格优于 excluded」成立，且与那扇窗是不是 w02 无关。**
